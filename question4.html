Position : Static -> If an element is defined as static, then the element will always have the position as per the normal page flow. This is the condition when nothing is defined as an element in an HTML page design. It is also the default value. The static element does not change its position.

______________________________________________________________________________________________________________________________________________

Position : relative ->(on box2) If an element is defined as relative, then the element will always have the position according to the defined values of its position like: left, right, top, bottom relative to the normal position, and its movement is in liaise with scrolling of the page.
